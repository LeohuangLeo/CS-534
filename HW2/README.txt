Programing language: Python 3.7

Perceptron & Kernel Method

The source code contain 6 files:
main_part1.py : Online Perceptron 
main_part2.py : Average Perceptron
main_part3.py : Polynomial Kernel Perceptron, including data prediction (kplabel.csv)
Read_data.py  : Read training and validation data
Read_test_data.py : Read testing data
Data_pred_op_ap.py : data prediction for Online Perceptron (oplabel.csv) and Average Perceptron (aplabel.csv)

Source datasets:
pa2_train.csv (file size is larger than 25MB, so I cannot upload to here. However, user can split valid data into train and val as well)
pa2_valid.csv
pa2_test_no_label.csv

Testing datasets:
oplabel.csv
aplabel.csv
kplabel.csv

Instruction:
The user should run main_file.py (main_part1.py, main_part2.py, main_part3.py) step by step. Each main code shows and creates the corresponding result and testing file to the different scenario.
