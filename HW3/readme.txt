Programing language: Python 3.7

Decision Tree Ensemble for Mushroom Classification

The 8 source codes as follow:
main_part1.py : Decision tree (DT)
main_part2.py : Random Forest (Bagging)
main_part3.py : AdaBoost (Boosting)
test_pred.py : Test file prediction
Data_prep.py : Data preprocessing
Decision_tree.py : Decision tree algorithm
Random_forest.py : Random forest algorithm
Adaboost.py : Adaboost algorithm

Source datasets:
pa3_train.csv
pa3_val.csv
pa3_test.csv

Generate dataset:
pa3_test_pred.csv

Instruction:
The user should run main_file.py (main_part1.py, main_part2.py, main_part3.py) and test_pred.py step by step. Each main code shows and creates the corresponding result. test_pred.py can generate test file named ‘pa3_test_pred.csv’.
